📊 Analytics

This section is where you'll track your social media performance and analyze your results.

Table Properties

| Property | Type | Description | | --- | --- | --- | | Date | Date | The date of the post | | Platform | Select | The social media platform | | Engagement | Number | The number of likes, comments, and shares | | Reach | Number | The number of people who saw the post | | Impressions | Number | The number of times the post was displayed | | Clicks | Number | The number of clicks on the post | | Conversions | Number | The number of conversions (e.g. sales, sign-ups) |

Formula Suggestions

* Calculate the engagement rate: `Engagement / Reach * 100` * Calculate the click-through rate: `Clicks / Impressions * 100` * Calculate the conversion rate: `Conversions / Clicks * 100`
