Social Media Content Planner Canva

How to Use 📚

This Social Media Content Planner Canva template is designed to help you organize and schedule your social media content in advance. It's perfect for small business owners, marketers, and influencers who want to save time and increase their online presence. To get started, simply import this template into your Notion account and follow the instructions below.

Introduction to the Template 📊

This template consists of several sections, each with its own database/table and properties. The main sections are:

* Content Calendar: A calendar view of all your scheduled posts * Content Library: A database of all your social media content, including images, videos, and captions * Hashtag Research: A database of relevant hashtags for your niche * Influencer Collaboration: A database of influencers you've collaborated with or want to collaborate with * Analytics: A section to track your social media performance and analytics

Content Calendar 📆

Table Headers and Properties

| Property | Type | Description | | --- | --- | --- | | Date | Date | The date of the scheduled post | | Platform | Select | The social media platform (e.g. Facebook, Instagram, Twitter) | | Content Type | Select | The type of content (e.g. image, video, caption) | | Caption | Text | The caption for the post | | Image/Video | File | The image or video file for the post | | Hashtags | Multi-Select | The relevant hashtags for the post | | Engagement | Number | The number of likes, comments, and shares for the post |

Formula Suggestions

* Use the `=DateAdd(Date, 7, "days")` formula to schedule a post for the same day next week * Use the `=Propagate(Platform, "Facebook")` formula to automatically set the platform for a post

Example Rows

| Date | Platform | Content Type | Caption | Image/Video | Hashtags | Engagement | | --- | --- | --- | --- | --- | --- | --- | | 2023-02-20 | Instagram | Image | "New product launch!" | product-image.jpg | #newproduct #launch | 100 | | 2023-02-22 | Twitter | Caption | "Did you know that..." | | #didyouknow #funfact | 50 | | 2023-02-25 | Facebook | Video | "Check out our new video!" | video.mp4 | #newvideo #facebook | 200 |

Usage Instructions

1. Create a new row for each scheduled post 2. Fill in the date, platform, content type, caption, image/video, and hashtags 3. Use the formula suggestions to automate repetitive tasks 4. Track engagement metrics for each post to see what's working and what's not

Content Library 📚

Table Headers and Properties

| Property | Type | Description | | --- | --- | --- | | Title | Text | The title of the content | | Description | Text | A brief description of the content | | Image/Video | File | The image or video file | | Category | Select | The category of the content (e.g. product, blog post, quote) | | Tags | Multi-Select | Relevant keywords for the content |

Formula Suggestions

* Use the `=Filter(Content Library, Category, "product")` formula to show only product-related content * Use the `=Propagate(Tags, "newproduct")` formula to automatically add a tag to new product content

Example Rows

| Title | Description | Image/Video | Category | Tags | | --- | --- | --- | --- | --- | | Product Image | Image of our new product | product-image.jpg | Product | #newproduct #productimage | | Blog Post | Summary of our latest blog post | blog-post-summary.pdf | Blog Post | #blogpost #marketing | | Quote | Inspirational quote for the day | quote-image.jpg | Quote | #quote #inspiration |

Usage Instructions

1. Create a new row for each piece of content 2. Fill in the title, description, image/video, category, and tags 3. Use the formula suggestions to automate content organization 4. Use the content library to plan and schedule content in advance

Hashtag Research 📊

Table Headers and Properties

| Property | Type | Description | | --- | --- | --- | | Hashtag | Text | The hashtag | | Category | Select | The category of the hashtag (e.g. product, niche, industry) | | Popularity | Number | The popularity of the hashtag (e.g. number of uses) | | Relevance | Number | The relevance of the hashtag to your niche |

Formula Suggestions

* Use the `=Filter(Hashtag Research, Category, "product")` formula to show only product-related hashtags * Use the `=Propagate(Popularity, ">1000")` formula to show only popular hashtags

Example Rows

| Hashtag | Category | Popularity | Relevance | | --- | --- | --- | --- | | #newproduct | Product | 1000 | 8/10 | | #marketing | Niche | 5000 | 9/10 | | #industrynews | Industry | 2000 | 7/10 |

Usage Instructions

1. Research and add relevant hashtags to the database 2. Use the formula suggestions to filter and prioritize hashtags 3. Use the hashtag research to plan and schedule content with relevant hashtags

Influencer Collaboration 🤝

Table Headers and Properties

| Property | Type | Description | | --- | --- | --- | | Influencer | Text | The name of the influencer | | Niche | Select | The niche of the influencer (e.g. beauty, fitness, travel) | | Reach | Number | The reach of the influencer (e.g. number of followers) | | Engagement | Number | The engagement of the influencer (e.g. likes, comments) |

Formula Suggestions

* Use the `=Filter(Influencer Collaboration, Niche, "beauty")` formula to show only beauty influencers * Use the `=Propagate(Reach, ">10000")` formula to show only influencers with a large reach

1. Research and add relevant influencers to the database 2. Use the formula suggestions to filter and prioritize influencers 3. Use the influencer collaboration database to plan and schedule collaborations
