Brand Kit Canva Template

How to Use 📚

This Brand Kit Canva Template is designed to help you organize and manage your brand's visual identity. It includes a database to store your brand's assets, a section for color palettes, typography, and imagery, as well as a template for creating a brand style guide. To get started, simply import this template into your Notion account and follow the instructions below.

Brand Assets Database 🗂️

This database stores all of your brand's visual assets, including logos, icons, and graphics.

Table Headers and Properties

| Property | Type | Description | | --- | --- | --- | | Asset Name | Text | The name of the asset | | Asset Type | Select | The type of asset (e.g. logo, icon, graphic) | | File | File | The actual file for the asset | | Description | Text | A brief description of the asset | | Tags | Multi-Select | Relevant tags for the asset (e.g. color, shape, theme) |

Formula Suggestions

To automatically generate a preview of the asset, you can use the following formula: `=if(prop("File") != "", "Preview: " + prop("File"), "No file uploaded")`

Example Rows

| Asset Name | Asset Type | File | Description | Tags | | --- | --- | --- | --- | --- | | Logo | Logo | logo.png | The main logo for the brand | primary, color | | Icon | Icon | icon.png | A small icon for social media | secondary, monochrome | | Graphic | Graphic | graphic.png | A graphic for blog posts | illustration, theme |

Color Palettes 🎨

This section stores your brand's color palettes, including the hex codes and descriptions.

Table Headers and Properties

| Property | Type | Description | | --- | --- | --- | | Palette Name | Text | The name of the color palette | | Primary Color | Text | The primary color of the palette (hex code) | | Secondary Color | Text | The secondary color of the palette (hex code) | | Accent Color | Text | The accent color of the palette (hex code) | | Description | Text | A brief description of the palette |

Formula Suggestions

To automatically generate a color swatch, you can use the following formula: `=if(prop("Primary Color") != "", "Primary Color: " + prop("Primary Color"), "No primary color set")`

Brand Style Guide 📄

This template provides a basic structure for creating a brand style guide.

Usage Instructions

To create a brand style guide, simply fill out the template with your brand's information and assets. You can then export the guide as a PDF or share it with team members and stakeholders.

Example Content

* Introduction to the brand and its mission * Overview of the brand's visual identity, including logos, color palettes, and typography * Guidelines for using the brand's assets, including imagery and tone of voice * Examples of the brand's style in action, including website screenshots and social media posts

Conclusion 🎉

This Brand Kit Canva Template is designed to help you create a comprehensive brand style guide and manage your brand's visual assets. By following the instructions and using the template, you can ensure consistency across all of your brand's touchpoints and create a strong visual identity.
